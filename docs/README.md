# Документация проекта 50hz-spwm

Эта папка содержит документацию для проекта генератора SPWM на частоте 50 Гц.

## Содержание

- [github-workflow.md](github-workflow.md) - Подробное руководство по работе с Git и GitHub для этого проекта

## О проекте

**50hz-spwm** - проект для STM32F401CCU6 микроконтроллера.

### Технические характеристики

- **Микроконтроллер**: STM32F401CCU6
- **Частота системы**: 84 МГц
- **Внешний кварц**: 25 МГц
- **RAM**: 64 КБ
- **Flash**: 256 КБ

### Структура проекта

```
50hz-spwm/
├── Core/               # Исходный код приложения
│   ├── Inc/           # Заголовочные файлы
│   └── Src/           # Исходные файлы
├── Drivers/           # HAL драйверы STM32
├── cmake/             # CMake конфигурация
├── docs/              # Документация
├── build/             # Сборка (игнорируется git)
├── CMakeLists.txt     # Основной файл CMake
└── 50hz-spwm.ioc     # Файл конфигурации CubeMX
```

### Сборка проекта

```bash
# Конфигурация
cmake -B build/Debug --preset=Debug

# Сборка
cmake --build build/Debug -j$(nproc)

# Очистка и пересборка
rm -rf build/Debug && cmake -B build/Debug --preset=Debug && cmake --build build/Debug -j$(nproc)
```

### Вывод бинарных файлов

После сборки файлы находятся в `build/Debug/`:
- `50hz-spwm.elf` - исполняемый файл с отладочной информацией
- `50hz-spwm.bin` - бинарный файл для прошивки (если настроено)
- `50hz-spwm.hex` - HEX файл для прошивки (если настроено)
